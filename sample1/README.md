# Sample1 - Python to Scratch

Python : send broadcast message <br/>
Scratch : move the cat, when recieve broadcast message <br/>

Message is a letter of the alphabet.
- c: to move to the center
- l: to move to the left
- r: move to the right
- u: to move up
- d: move down
